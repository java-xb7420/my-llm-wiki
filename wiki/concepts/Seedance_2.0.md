# Seedance 2.0

## 概述

Seedance 2.0 是火山引擎推出的视频生成模型，于 2026 年 4 月全面开放 API 服务，面向企业和个人开放视频生成能力。

## 核心能力

### 四模态输入
- **文本** - 文字描述生成视频
- **图片** - 图片转视频
- **音频** - 音频驱动视频生成
- **视频** - 视频编辑和转换

### 多模态参考
- 支持内容参考
- 风格参考
- 多参考源融合

### 编辑能力
- 视频元素编辑
- 风格迁移
- 内容修改

### 版权安全
- 建立肖像版权安全标准
- 人脸识别和保护
- 版权内容过滤

## 行业应用

| 行业 | 应用案例 |
|------|----------|
| 影视文娱 | 贾樟柯导演短片、春晚应用 |
| 漫短剧 | 短剧内容生成 |
| 广告营销 | 广告视频快速制作 |
| AI 工具 | 集成到第三方 AI 产品 |
| 具身智能 | 机器人训练数据生成 |
| 自动驾驶 | 驾驶场景模拟 |

## API 服务

### 计费方式
- 按调用量计费

### 接入方式
