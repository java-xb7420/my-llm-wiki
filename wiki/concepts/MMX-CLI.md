# MMX-CLI

## 概述

MMX-CLI 是 MiniMax 推出的面向 AI Agent 的命令行工具，于 2026 年 4 月发布。工具支持在 Claude Code、OpenClaw 等环境原生调用 MiniMax 全模态模型。

## 核心特性

### Agent 优化设计

| 特性 | 描述 |
|------|------|
| 输出隔离 | Agent 输出与人类可读输出分离 |
| 语义化状态码 | 机器可读的状态反馈 |
| 异步任务控制 | 长任务的启动、查询、取消 |

### 全模态支持

无需适配接口或编写 MCP Server，即可调用：
- 编程模型
- 视频生成
- 语音合成
- 音乐创作

### 自动化工作流

可独立跑完整工作流：
```
资料搜集 → 文案生成 → 语音合成 → 视频制作
```

## 兼容环境

- **Claude Code** - Anthropic CLI 工具
- **OpenClaw** - 开源 AI 开发工具
- 其他支持 CLI 的 AI Agent 环境

## 使用场景

| 场景 | 描述 |
|------|------|
| 视频制作 | 自动化视频生成流程 |
| 播客制作 | 文案 + 语音合成 |
| 营销内容 | 图文 + 视频内容生成 |
| 代码开发 | 调用编程模型辅助 |

## 技术优势

### 零适配
- 无需编写 MCP Server
- 无需适配 API 接口
- 开箱即用

### Agent 友好
- 结构化输出
- 状态码反馈
- 异步任务支持

### 多模态集成
- 统一命令行接口
- 多模型协同
- 资源包优化
