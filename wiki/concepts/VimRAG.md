# VimRAG

## 概述

VimRAG 是阿里通义实验室推出的全模态知识库 RAG（检索增强生成）框架，于 2026 年 4 月开源。模型用动态有向无环图（DAG）替代传统的线性上下文拼接。

## 核心创新

### DAG 架构
- **动态有向无环图** - 替代线性上下文拼接
- **分支试错机制** - 多路径检索尝试
- **视觉能量分配** - 跨模态注意力分配
- **图引导优化 (GGPO)** - 基于图结构的检索优化

### 跨模态能力
- **文本检索** - 文档、文章检索
- **图像检索** - 图片内容理解和检索
- **视频检索** - 视频内容检索
- **跨模态关联** - 不同模态间的关联推理

## 性能表现

在 Qwen3-VL-8B 模型上的测试结果：

| 指标 | VimRAG | 传统 RAG |
|------|--------|----------|
| 准确率 | 50.1% | ~30% |
| 跨模态检索 | 显著提升 | 基础 |
| 关联推理 | 强 | 弱 |

## 技术架构

```
输入 → DAG 索引 → 分支检索 → 能量分配 → GGPO 优化 → 输出
         ↓           ↓           ↓           ↓
    文本/图像/视频   多路径    注意力分配   图引导排序
```

## 应用场景

| 场景 | 描述 |
|------|------|
| 企业知识库 | 跨文档、图片、视频的综合检索 |
| 科研文献 | 论文、图表、数据的关联检索 |
| 多媒体内容 | 图文混排、视频内容检索 |
| 智能问答 | 基于多源信息的精准回答 |

## 与传统 RAG 对比

| 特性 | 传统 RAG | VimRAG |
|------|----------|--------|
| 索引结构 | 向量索引 | 动态 DAG |
| 上下文拼接 | 线性 | 图结构 |
| 检索路径 | 单一 | 多分支试错 |
| 模态支持 | 单模态为主 | 全模态 |
