# Agent智能体

## 概述

Agent智能体是能自主规划、调用工具、执行复杂任务的AI实体。2026年3-4月，Agent从对话助手进化为可7×24小时自主运行的"龙虾"形态，标志着AI从"聊天工具"向"自主实体"的范式转变。

## 核心能力

- **自主规划**：分解复杂任务，制定执行策略
- **工具调用**：操作电脑、浏览器、API等外部工具
- **持续运行**：云端沙箱7×24小时后台执行
- **长期记忆**：跨会话、跨平台的个性化记忆系统
- **多Agent协作**：Agent Teams编排与协调

## 自我进化Agent

[[MiniMax M2.7]] 率先实现"模型深度参与自我迭代"，通过：
- 学习闭环：完成任务后自动提炼可复用Skills
- Agent Harness：构建复杂Agent工作流
- Agent Teams：多Agent协同完成生产力任务
