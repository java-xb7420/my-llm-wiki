# ERNIE-Image

## 概述

ERNIE-Image 是百度文心推出的文生图模型，于 2026 年 4 月开源。模型仅 8B 参数，在文字渲染、复杂指令遵循等能力上媲美 Nano Banana 等商业闭源模型。

## 技术规格

| 规格 | 数值 |
|------|------|
| 参数量 | 8B |
| 显存需求 | 24GB |
| 支持语言 | 中文、英文、日文、韩文 |
| 开源协议 | Apache 2.0 |

## 核心能力

### 文字渲染
- 精准字形生成
- 支持多语言文字混合
- 复杂排版处理

### 复杂指令遵循
- 多元素组合生成
- 空间关系理解
- 风格迁移控制

### 多语言支持
- 中/英/日/韩四语支持
- 跨语言风格迁移
- 方言变体生成

## 部署方案

### ComfyUI 集成
已上线 ComfyUI，支持可视化工作流编排

### GGUF 量化
推出 GGUF 量化方案，降低推理资源需求

### 开源地址
- Hugging Face 权重已开源
- 推理代码已开源

## 竞品对比

| 模型 | 参数量 | 开源 | 显存需求 |
|------|--------|------|----------|
| ERNIE-Image | 8B | ✓ | 24GB |
| Nano Banana | 闭源 | ✗ | - |
| Stable Diffusion | 1B-4B | ✓ | 8-16GB |

## 应用场景

- 商业设计 - 广告图、海报设计
- 电商 - 商品图生成
- 社交媒体 - 配图生成
- 教育 - 教学素材制作
