# VoxCPM2

## 概述

VoxCPM2 是 OpenBMB 开源的 2B 参数语音合成模型，于 2026 年 4 月发布。模型采用无分词器扩散自回归架构，支持 30 种语言及中文方言，输出 48kHz 录音室级音质。

## 技术规格

| 规格 | 数值 |
|------|------|
| 参数量 | 2B |
| 支持语言 | 30 种 + 中文方言 |
| 输出音质 | 48kHz 录音室级 |
| 训练数据 | 236 万小时 |
| 开源协议 | Apache 2.0 |

## 核心创新

### 无分词器架构
- **扩散自回归** - 结合扩散模型和自回归优势
- **无分词器** - 避免分词信息损失
- **端到端** - 直接从文本到音频

### Voice Design 功能
- **凭空创造声音** - 通过文字描述创造新声音
- **声音参数控制** - 精确控制声音特征
- **风格迁移** - 声音风格转换

### 可控声音克隆

| 模式 | 描述 |
|------|------|
| 可控克隆 | 控制声音特征进行克隆 |
| 终极克隆 | 高保真声音克隆 |
