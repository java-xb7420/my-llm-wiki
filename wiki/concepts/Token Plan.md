# Token Plan

## 概述

Token Plan是2026年AI行业兴起的新型订阅计费模式，采用统一Credit点数计费，替代传统的按token用量或按时长限制计费。

## 代表方案

| 方案 | 公司 | 档位 |
|------|------|------|
| MiMo Token Plan | 小米 | Lite ¥39/月、Standard ¥99/月、Pro ¥329/月、Max ¥659/月 |
| MiniMax Token Plan | MiniMax | 全模态支持，语音/视频资源包省20% |

## 核心创新

- **统一Credit**：一个Key满足代码、视频、语音、音乐等多模态需求
- **取消时长限制**：废除5小时token使用限额
- **适配主流工具**：兼容Claude Code、OpenClaw等AI开发工具
- **资源包优化**：针对高频使用场景推出折扣资源包

## 背景

传统API按token计费对高频用户不友好，且不同模型/模态的计费规则各异。Token Plan通过统一计费简化用户决策，同时提高ARPU。
