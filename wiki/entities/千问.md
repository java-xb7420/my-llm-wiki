# 千问

## 概述

千问是阿里巴巴推出的全能型人工智能助手，基于通义千问（Qwen）大模型，具备[[全模态大模型]]能力，支持文本、图片、文档、音频和视频等多种输入方式，为用户提供一站式智能服务。

## 核心能力

### 多模态交互
- 支持文本、图片、文档、音频和视频等多种输入方式
- 能够对不同格式的内容进行理解和处理

### 长文本处理
- 单次可处理多达 1000 万字的长文档
- 支持同时解析 100 份不同格式的文件
- 适合处理复杂的学术论文、报告等

### 智能写作
- 提供创意文案生成、办公文档撰写、学习辅导等多种服务
- 能够撰写故事、公文、邮件、剧本和诗歌等

### 实时翻译
- 支持多达 119 种语言和方言的翻译与对话
- 帮助用户跨越语言障碍

### 音视频处理
- 能对音视频文件进行识别、理解、摘要总结和多语言翻译
- 支持同时上传处理 50 个音视频文件

## 特色功能

| 功能 | 描述 |
|------|------|
| PPT 创作 | 支持 10 万字超长文本的 PPT 生成，并提供 pptx 文件下载 |
| 口语练习 | 支持多种语言实时对话，提供发音、语法和流利度反馈（仅手机 APP） |
| AI 证件照 | 快速生成符合不同场景要求的证件照 |
| AI 试衣 | 利用图像处理技术，虚拟试穿各种服装 |
| AI 生视频 | 包括文生视频和图生视频两种方式 |
| AI 修图 | 支持扩图、风格转换、去水印等功能 |
| 实时记录 | 支持多语言翻译，内容重点总结，可生成思维导图 |
| 通义灵码 | 支持代码生成、优化、注释、解释、研发智能问答、报错排查 |
| 通义听悟 | 语音实时转换为文字，进行总结，生成思维导图 |

## 使用方式

- **网页版**：访问通义官网 https://www.qianwen.com/
- **手机应用**：应用商店搜索"千问"下载安装
- **微信小程序**：微信小程序搜索"千问"
- **浏览器插件**：适用于网课、追剧、线上会议等场景

## 应用场景

- **会议管理**：实时转录会议内容，自动生成中英双语纪要
- **文档处理**：快速生成市场调研报告、项目总结等文档
- **智能写作**：自动生成文章、报告、邮件等高质量文本
- **学习辅导**：为学生提供个性化学习指导，解答疑问
- **设备控制**：通过语音或文字指令控制家用智能设备
