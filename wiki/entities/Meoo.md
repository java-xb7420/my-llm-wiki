# Meoo

## 概述

Meoo（秒悟）是阿里 ATH 事业群推出的首款 AI 开发工具，于 2026 年 4 月发布。用户无需编程基础，通过自然语言描述需求，最快 1 分钟可自动生成完整前后端代码。

## 核心特性

### 多模型集成
集成四大顶尖模型：
- [[Qwen|千问]] - 阿里通义千问
- [[Kimi]] - 月之暗面
- [[GLM]] - 智谱 AI
- [[MiniMax]] - 稀宇智能

### 云服务集成
内置阿里云核心服务：
- 阿里云数据库
- 云存储服务
- 其他阿里云 PaaS 服务

### 开发流程

```
自然语言描述 → AI 理解需求 → 生成前后端代码 → 阿里云一键部署
           ↓
         1 分钟完成
```

## 目标用户

- 非技术背景的业务人员
- 快速原型开发者
- 创业团队
- 企业内部工具开发者

## 使用场景

| 场景 | 描述 |
|------|------|
| 业务工具 | 快速搭建内部管理系统 |
| 数据展示 | 搭建数据可视化看板 |
| API 服务 | 快速构建 RESTful API |
| 网站搭建 | 企业官网、落地页生成 |

## 部署方式

- **阿里云一键部署** - 生成代码直接部署到阿里云
- **本地开发** - 支持本地调试和运行
- **容器化部署** - 支持 Docker 容器部署
